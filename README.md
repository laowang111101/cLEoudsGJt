# 课程作业管理系统简介 java799

## 项目概述

课程作业管理系统是一款旨在简化和优化课程作业流程的教育管理软件。系统通过整合学生、教师、课程、作业等管理功能，提高教学管理的效率与质量。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js + Element UI
- 数据库：MySQL
- 缓存：Redis
- 运行环境：JDK 1.8 + Tomcat 8
- Node.js（用于前端构建）

## 功能模块

- **登录模块**：用户身份认证
- **学生注册**：学生信息注册
- **个人中心**：查看与编辑个人信息
- **公告管理**：发布、查看、编辑公告
- **学生管理**：管理学生信息
- **教师管理**：管理教师信息
- **课程管理**：课程信息增删改查
- **学生选课管理**：学生选课操作
- **作业布置**：教师发布作业
- **作业提交**：学生提交作业
- **作业评分管理**：作业评分与反馈
- **课程评价管理**：课程评价收集
- **课程资源管理**：课程资源上传与分享

## 系统角色

- 学生：选课、提交作业、评价课程
- 教师：管理课程、布置作业、评分作业、发布公告
- 管理员：管理用户信息、课程信息、公告等

## 运行环境

- JDK 1.8
- IntelliJ IDEA
- MySQL 5.7+
- Redis 3.2+
- Tomcat 8

## 部署步骤

1. 准备运行环境：安装 JDK 1.8、MySQL、Redis、Tomcat 8。
2. 构建后端项目：使用 Spring Boot 打包应用程序。
3. 构建前端项目：使用 Vue.js 和 Element UI。
4. 部署前端项目到 Web 服务器。
5. 部署后端项目到应用服务器。
6. 启动服务并测试。
